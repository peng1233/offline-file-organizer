# Offline File Organizer（全离线）服务描述（版本B：更短更销售）

一句话：我会交付一个 **完全离线运行** 的网页小工具，帮你把一堆文件按规则重命名/整理，并导出 **CSV 清单**（用于归档、对账、交付）。

## 你得到什么
- 一个可直接打开使用的 HTML 工具页（不联网也能用）
- 规则化重命名/批量预览/一键应用
- 可选：ZIP 打包导出（环境不支持时自动提示并给出 Lite 兜底）
- CSV 清单导出（文件名/路径/大小/时间等）

## 适合谁
- 素材/视频/照片：整理命名 + 给客户一份清单
- 项目交付/审计留档：按约定规则统一命名
- 任何“不想上传文件到云端”的场景（隐私/合规）

## 为什么可信
- 100% 本地处理：文件不上传、不会走服务器
- 有 Lite 兜底：就算 ZIP 组件不可用也能用核心功能
