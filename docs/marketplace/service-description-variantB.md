# Offline File Organizer — Service Description (Variant B)

> Positioning: “privacy-first, offline-first, fast turnaround”, with a clearer scope and acceptance checklist.

## English (Variant B)

I build a **privacy-first, offline-first** file renaming & packaging tool that runs locally in your browser (no uploads). It helps you **standardize filenames**, produce a **CSV manifest**, and optionally **download as ZIP** (with a Lite fallback if ZIP isn’t supported).

**Typical use cases**
- Media teams: rename photo/video assets consistently
- Admin/ops: organize invoices/receipts/contracts for delivery
- Secure environments: intranet/offline workflows where uploads are not allowed

**What you get**
- Offline web app (HTML) + concise usage guide
- CSV export (for audit/tracking)
- ZIP download (when available) + fallback one-by-one download

**Customization options (if needed)**
- Naming templates (prefix/date/sequence)
- Duplicate/conflict rules
- CSV columns and ordering
- Preset buttons for common workflows

**Acceptance checklist**
- Works offline after you open the HTML file
- Renames apply to downloaded outputs (original files remain unchanged)
- CSV export matches agreed columns
- ZIP works when supported; otherwise fallback download still works

- Lite fallback: https://peng1233.github.io/offline-file-organizer/lite/

Repo: https://github.com/peng1233/offline-file-organizer

## 中文（Variant B）

我可以交付一个 **隐私优先/离线优先** 的文件整理工具：在本地浏览器中批量改名、导出 CSV 清单，并可选 ZIP 打包下载（若环境不支持 ZIP，会自动使用 Lite 兜底的逐个下载方式）。

**可选定制**
- 命名模板（前缀/日期/序号）
- 重名冲突规则
- CSV 字段与顺序
- 常用规则预设按钮
