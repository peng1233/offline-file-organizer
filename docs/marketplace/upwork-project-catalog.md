# Upwork Project Catalog（可复制粘贴模板）

> 用途：把这个文件里的内容直接复制到 Upwork Project Catalog 页面里即可。
> 说明：不承诺夸大效果；交付以“可运行 + 可验收”为准。

## 项目标题（建议）
**Offline File Organizer（100%离线的批量重命名 + CSV清单导出 + 可选ZIP打包）**

## 一句话卖点
一个无需安装、在浏览器里离线运行的小工具：批量重命名文件，并导出 CSV 清单，必要时可打包 ZIP（不支持 ZIP 的环境可用 Lite 兜底）。

## 适用人群
- 需要整理图片/视频素材、交付物、发票收据等的个人或团队
- 注重隐私或内网环境：不允许上传到网盘/服务器
- 想把“交付清单/对账/归档”标准化的人

---

## 套餐（建议三档）

### BASIC（交付现成可用 + 轻量配置）
**你会得到**
- 可离线使用的工具（HTML + 简短说明）
- 1 套命名模板（例如：`Project_YYYYMMDD_###`）
- CSV 清单导出字段保持默认

**验收标准**
- 你提供的样例文件可被批量重命名
- 可导出 CSV 清单（能用 Excel/表格软件打开）

### STANDARD（按你的规则定制）
**你会得到**
- BASIC 全部内容
- 2~3 套命名模板 / 预设按钮
- 冲突处理策略按你的习惯调整（例如：自动加序号/阻止覆盖/提示确认）
- CSV 清单导出字段可按需增删（例如：原名/新名/大小/日期/路径）

**验收标准**
- 规则在你提供的样例数据上跑通
- 输出 CSV 字段符合约定

### PREMIUM（交付级体验 + 演示素材）
**你会得到**
- STANDARD 全部内容
- 可选：ZIP 打包下载（如环境不支持会保留 Lite 兜底）
- 1 份“截图/短GIF”演示素材（便于你内部培训或对外交付）

**验收标准**
- 主流程（导入→重命名→导出清单→下载）跑通
- Lite 兜底模式可用（至少支持：重命名 + CSV 导出 + 逐个下载）

---

## 你需要提供（开始前）
- 3~10 个示例文件名（可匿名/可打码）
- 你想要的命名规则（示例最好）
- 是否需要 ZIP（如果你环境经常禁用 ZIP，直接走 Lite 也没问题）

## 风险与边界（我会提前说明）
- 不同浏览器对 ZIP/下载能力支持差异较大：会提供 Lite 兜底
- 不涉及任何服务器端上传/存储（工具默认 100% 离线本地）
