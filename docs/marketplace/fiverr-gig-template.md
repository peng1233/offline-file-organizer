## 3) Gig Description（正文，可直接粘贴）

I deliver a **privacy-first, fully offline** file organizing / batch renaming workflow.

What you get:
- Batch rename rules (prefix/suffix/replace/numbering)
- Preview before you apply
- Export a **CSV manifest** (for review/approval/audit)
- Optional ZIP packaging, plus a **Lite fallback** that works with zero third‑party libraries
- Windows-friendly quick start (PowerShell one-command / localhost helper)

Why offline:
- No uploading files
- Works in air‑gapped environments
- Faster iteration and easier compliance

Proof & assets:
- Repo: https://github.com/peng1233/offline-file-organizer
- Live demo (Lite): https://peng1233.github.io/offline-file-organizer/lite/

## 6) FAQ（常见问题）

Q: Do you need my files?
- No. You can share anonymized filenames only.

Q: Can it work without internet?
- Yes. The Lite version is designed to be fully offline.

Q: Will it modify my original files?
- No. It generates a plan and downloads renamed copies.

Q: Browser compatibility?
- Works best on modern Chromium browsers. Lite includes compatibility fallbacks.
