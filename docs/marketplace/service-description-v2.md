## 中文（版本 2）

我可以为你交付一套 **完全离线、本地运行、可审计** 的文件整理小工具（浏览器打开即可用）：
- 批量重命名文件（按模板/序号/日期等规则）
- 导出文件清单（CSV），方便对账/交付/归档
- （可选）将结果打包为 ZIP 下载
- 当 ZIP 在某些浏览器环境不可用时，仍可通过 **Lite 纯离线兜底方案** 逐个下载文件，保证流程不中断

**适合人群 / 场景**
- 内网/离线环境：不能安装软件、不能联网、不能上传文件
- 隐私敏感资料：合同、发票、项目交付、客户资料（仅在本地处理）
- 需要可追溯交付：要一份 文件清单作为交付/验收依据

**你将获得什么**
- 一套可直接交付给同事/客户使用的离线网页工具（HTML + 说明）
- 一份可复现的命名规则说明（避免口头规则导致反复沟通）
- （可选）按你的行业习惯定制：命名模板、冲突处理、CSV 列、默认按钮/预设

**我需要你提供什么**
- 少量示例文件名/目录结构（可脱敏）
- 目标命名规则样例（例如：Client_Project_YYYYMMDD_###）

## English (Version 2: privacy / offline / auditable delivery)

I can deliver a **fully offline, local-first, auditable** file organizing tool (open in a browser, no installation):
- Batch rename files (templates / sequence / dates)
- Export a manifest (CSV) for tracking, delivery, and auditing
- (Optional) download everything as a ZIP
- If ZIP isn’t available in your environment, it still works via a **no-dependency Lite fallback** (download files one-by-one) so your workflow never blocks

**Great for**
- Intranet/offline workflows (no installs, no internet, no uploads)
- Privacy-sensitive materials (contracts, invoices, deliverables)
- Teams who need an auditable manifest as part of acceptance/delivery

**Deliverables**
- A ready-to-use offline web tool (HTML + docs)
- A reproducible naming-rule spec (reduces back-and-forth)
- (Optional) customization: naming templates, conflict rules, CSV columns, presets

**What I need from you**
- A few sample filenames/folder structure (can be anonymized)
- Your target naming rules (e.g., Client_Project_YYYYMMDD_###)
