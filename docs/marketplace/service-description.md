## 中文（建议用于 Upwork / Fiverr 简介）

我可以为你交付一套 **完全离线、在浏览器本地运行** 的文件整理小工具：批量重命名文件、导出文件清单（CSV），并支持 **可选 ZIP 打包下载**。
如果你的环境里 ZIP 能力不可用（或不稳定），也可以使用 **Lite 纯离线兜底版本**（逐个下载），仍然能完成文件整理与交付。

**适用场景**
- 图片/视频素材批量整理（日期 / 序号 / 模板命名）
- 文档归档与交付（合同、发票/票据、项目交付文件等）
- 隐私/内网环境（不能上传到网盘或第三方服务）

**功能要点**
- **100% 本地离线**：不上传文件到任何服务器
- **可选 ZIP**：支持一键打包；不支持时自动有 Lite 兜底
- **导出 CSV 清单**：用于对账、交付、归档
- **零安装**：打开 HTML 页面即可使用

**交付内容**
- 可直接使用的离线网页工具（HTML + 使用说明）
- （可选）定制：命名模板、冲突规则、导出列、预设按钮等

**你需要提供**
- 若干示例文件名/目录结构（可匿名）
- 你的目标命名规则（例如：`Project_YYYYMMDD_###`）

## English (for Upwork / Fiverr description)

I can deliver a **fully offline** file-organizing tool that runs in your browser: batch rename files, export a manifest (CSV), and optionally download everything as a ZIP.
If ZIP isn’t available in your environment, it still works via a **no-dependency Lite fallback** (download files one-by-one).

**Great for**
- Batch renaming photo/video assets (date / sequence / templates)
- Archiving documents (contracts, invoices/receipts, deliverables)
- Privacy-sensitive workflows (offline / intranet / no server uploads)

**Key features**
- **100% local & offline** (no uploads)
- **Optional ZIP** (with Lite fallback when ZIP isn’t supported)
- **CSV manifest export** for tracking and delivery
- **Zero install**: open the HTML page and use it

**Deliverables**
- A ready-to-use offline web tool (HTML + docs)
- (Optional) customization: naming templates, conflict rules, export columns, presets

**What I need from you**
- A few sample filenames / folder structure (can be anonymized)
- Your target naming rules (e.g., `Project_YYYYMMDD_###`)
