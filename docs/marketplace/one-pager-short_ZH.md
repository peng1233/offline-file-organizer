# 离线文件整理器（Offline File Organizer）— 中文一页说明（Upwork / Fiverr / Toptal）

**离线文件整理器**：在你的电脑本地（浏览器里）离线批量改名文件，并生成“旧名 -> 新名”的清单，支持导出/复制 CSV 或 TSV，方便审批、对账、交付与归档。

## 适用场景
- 你有一批文件需要统一命名规范（客户交付、资料归档、素材整理）
- 你需要可审计的改名前后映射表（CSV/TSV）用于确认与留档
- 你对隐私敏感（不上传、不出网），希望全程本地离线完成

## 你能得到什么（如果雇我交付）
1) 按你的命名标准调整规则（前后缀/替换/序号/日期等）
2) 做 1 个小增强（校验、预设、额外导出格式、UI 提示等）
3) 提供可复现 demo + 验收清单（小改动、可回滚）
