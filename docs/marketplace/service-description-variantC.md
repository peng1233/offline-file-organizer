# Offline File Organizer — Variant C (Short / Hard-hitting)

## EN (paste-ready)
**Offline File Organizer** helps you batch-rename files safely **without uploading anything**.

- 30 seconds to use: open the page, select files, set a rename pattern.
- **Privacy-first**: runs locally in your browser (no server).
- **Non-destructive**: never modifies your original files; you download renamed copies.
- Exports a **CSV manifest** for auditing.
- Includes a fully-offline **Lite** mode (no third-party dependencies).

Demo (Lite): https://peng1233.github.io/offline-file-organizer/lite/
Repo: https://github.com/peng1233/offline-file-organizer
Support / tip jar: https://peng1233.github.io/btc-receive/

**Deliverables (example)**
- A customized rename template/preset for your workflow
- A short usage guide + acceptance checklist
- Optional: add your company naming rules / conflict checks
