# Platform Keywords / Tags Cheat Sheet（平台关键词/标签速查）

> 目的：把“可复制粘贴”的关键词/标签一次性整理好，发布/投标时不再临时想词。
> 
> 说明：不包含任何账号信息/报价/收款引导；遵守平台规则（尤其 Upwork：不引导站外支付）。

---

## Core value props（通用卖点关键词）

- offline / fully offline / works offline
- privacy-first / local-only / no upload / no cloud
- batch rename / bulk rename
- file organizer / file renamer
- CSV export / rename plan / manifest
- Windows friendly / one-click / quick start
- browser tool / local web tool
