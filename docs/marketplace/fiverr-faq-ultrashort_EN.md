# Fiverr FAQ (ultra-short copy/paste)

All text is ASCII.
One sentence per answer for strict form limits.

1) Does this tool upload my files?
No, it runs locally in your browser and your files stay on your machine.

2) Do I need to install anything?
No, just open the HTML (or run localhost mode for best compatibility).

3) What do I receive after delivery?
An offline ZIP (HTML/CSS/JS) plus a short README and any agreed demo assets.

4) Can you customize the rename rules?
Yes, send examples of your naming rules and I will implement or adjust them.

5) Will it rename my original files?
No, it downloads renamed copies (or a ZIP) and never modifies originals.

6) Does it support CSV output?
Yes, it can export CSV and Lite can Copy CSV (with fallback if clipboard is blocked).

7) What browsers are supported?
Modern Chromium browsers work best; use localhost mode if file:// folder access is blocked.

8) Do you need remote access or my private files?
No, I can work from example filenames or a sample CSV, and remote access is optional.

9) Is it fully offline?
Yes, the Lite version is fully offline-capable (no third-party dependencies).

10) How many revisions are included?
Typically 1-2 revision rounds for naming rule tweaks (depends on the package).
