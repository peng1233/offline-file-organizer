# Fiverr / Upwork 服务包（定价 + 交付物清单）

> 目的：把“能卖什么、多少钱、交付什么”说清楚，方便你在 Fiverr/Upwork/Toptal 做上架与投递时直接复制粘贴。
>
> 适用产品：本仓库 **Offline File Organizer**（离线文件整理工具 + Lite 版本）

---

## 一句话卖点（可直接复制）

- I will deliver a privacy-first **offline file organizer** (web app) that helps you quickly sort and rename files with a clean UI.
- Works locally, no data upload. Includes a lightweight Lite version for quick preview.

---

## 服务包 1：Lite 快速交付（Starter）

**建议价格：US$15 ~ 30**

适合：客户想先试用、或只需要一个可离线打开的轻量工具。

交付物：
- 1 个可分享的 Lite ZIP：`offline-file-organizer-lite.zip`
  - 内含：HTML/CSS/JS（可直接双击打开）
  - 附带 `RELEASE.txt`（SHA256 + 文件清单）
- 1 份简短使用说明（中/英文二选一）

交付时间：24 小时内

需要客户提供：
- 目标用途（整理照片/文档/素材等）
- 是否需要自定义：默认目录规则/按钮文案/Logo（可选）

不包含：
- 复杂规则引擎
- 与云盘/第三方 API 集成

**建议价格：US$49 ~ 99**

交付物：

- 一次自检结果截图/日志（证明基本功能可用）
- 可选：在 README 中补 1 段“你的场景”的快速上手说明

交付时间：2 ~ 3 天

需要客户提供：
- 3~10 个样例文件名（或截图），说明“理想结果长什么样”

## 服务包 3：定制化工作流（Pro）

**建议价格：US$149 ~ 399**

适合：有明确业务流程（素材管理/剪辑项目/票据归档/课程资料等），需要定制 UI、规则、导入导出或批处理流程。

需要客户提供：

- 能在 Windows 10/11 的 Chrome/Edge 打开并正常使用
- 按约定规则对样例文件名输出符合预期的结果
- Lite ZIP 可解压后直接运行（无需安装）

## 风险与边界（避免扯皮）

- 若客户需要“读取/写入系统目录的高级能力”，建议通过 `http://localhost` 启动（浏览器权限限制决定）
- 复杂的第三方集成（Google Drive/Dropbox/OneDrive API）属于额外范围
